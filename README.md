# Advent of Code 2018

Each folder contains subfolders for the language(s) I've solved the day in

### Haskell

```shell
cd haskell
stack build --ghc-options -O2 --exec Day-%-exe
```
Where `%` is the day

### Node
```shell
node %
```
Where `%` is the filename

### C
Compile using gcc
```shell
gcc main.c -o main.o && ./main
```
